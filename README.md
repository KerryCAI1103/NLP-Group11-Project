情感匹配电影推荐系统 🎬
📋 项目简介
本项目实现了一个创新的电影推荐系统，能够将用户的情感语言表达与电影情感特征进行匹配。系统采用两种不同的技术方案：

LSA（潜在语义分析）框架 - 基于TF-IDF和SVD的传统自然语言处理技术
深度嵌入模型框架 - 结合Qwen3-Embedding和情感向量分析的现代深度学习方法
通过理解用户的情感描述（如"孤独但充满希望"、"紧张刺激"等），系统能够推荐情感氛围匹配的电影，而不仅仅是基于传统分类标签。

🚀 系统架构
graph LR
    A[原始电影文本] --> B[数据预处理]
    B --> C[TF-IDF向量化]
    C --> D[SVD降维]
    D --> E[潜在语义空间]
    E --> F[余弦相似度计算]
    G[用户查询] --> H[预处理&投影]
    H --> F
    F --> I[推荐结果]
技术特点：

使用jieba进行中文分词和停用词过滤
通过TF-IDF构建高维向量空间
应用SVD降维到20维潜在语义空间
基于余弦相似度进行电影匹配

方案二：深度嵌入模型框架
graph LR
    A[TMDB电影数据] --> B[情感分析]
    B --> C[10维情感向量]
    A --> D[Qwen3嵌入]
    D --> E[语义向量]
    C & E --> F[双通道向量空间]
    G[用户查询] --> H[情感分析]
    G --> I[语义编码]
    H & I --> J[混合相似度计算]
    J --> K[推荐结果]
技术特点：

爬取TMDB影史评分前250电影数据
自动情感分析生成10维情感向量（喜悦、悲伤、愤怒等）
使用Qwen3-Embedding模型进行语义理解
融合语义和情感相似度进行多维度推荐
